# Dieyun Song — Digital Portfolio

A minimalist, static portfolio site. Pure HTML + CSS, no build step.

## Pages
- `index.html` — Home / landing
- `cv.html` — Academic CV (fully populated)
- `resume.html` — Professional résumé (in progress)
- `projects.html` — Selected projects
- `contact.html` — Contact details

## Design
- **Palette (3 colors):** warm cream pastel background, charcoal ink, sage-green accent
- **Type:** Cormorant Garamond (headings) + Inter (body)
- Responsive and print-friendly (`@media print` styles the CV for PDF export)

## Local preview
Open `index.html` in a browser, or run a local server:

```bash
python3 -m http.server 8000
# then visit http://localhost:8000
```

## Deploy (GitHub Pages)
Settings → Pages → deploy from the `main` branch root. The site needs no build.
